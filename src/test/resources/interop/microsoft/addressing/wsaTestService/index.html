JBoss WS-Addressing 1.0 Service Endpoints

Overview

This service provides a JBossWS implementation of the service-side parts of the WS-Addressing 1.0 Test Suite.

Endpoints

The service is based on the Candidate Recommendation versions of WS-Addressing 1.0 - Core and WS-Addressing 1.0 - SOAP Binding.

  • wsaTestService

WSDL

The WSDL for the service can be found appending "?wsdl" to the service addresses, i.e.:

  • wsaTestService?wsdl
